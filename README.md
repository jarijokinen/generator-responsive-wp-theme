# generator-responsive-wp-theme

Responsive WordPress Theme Generator for Yeoman.

## Installation

### Yarn

    yarn global add yo
    yarn global add generator-responsive-wp-theme

## Usage

    mkdir my-theme
    cd my-theme
    yo responsive-wp-theme
